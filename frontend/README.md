## Local Development

**Run the backend first (required for `/tasks` data):**
```bash
npm run dev:watch -w backend
```

**Run the app locally:**
```bash
npm run dev
```

## Build

**Type-check and build for production:**
```bash
npm run build
```

**Preview the production build:**
```bash
npm run preview
```

## Lint

**Run ESLint:**
```bash
npm run lint
```
